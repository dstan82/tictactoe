# tictactoe
Tic Tac Toe game for 2 players

Requirments: 
Python 3 and Linux (due to clear screen function only works on linux)

Install:
$ git clone git@github.com:dstan82/tictactoe.git

Run:
$ python3 tictactoe/tictactoe.py

Play instructions: 
playing board is mapped to keys (1-9) according to the keyboard numpad. Both players take turns on the same computer.
